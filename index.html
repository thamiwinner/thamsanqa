About Me

My Software Development Journey

Foundations

My software development journey began with a Diploma in Business Information Technology, followed by a BCom in Information Systems at the University of Johannesburg. These programs laid a strong foundation in both technical skills and business acumen, crucial for navigating the tech industry.

Web Development Skills

Throughout my studies, I gained proficiency in various web development languages, including HTML, CSS, and JavaScript, essential for creating interactive and visually appealing websites. I also explored frameworks such as Bootstrap for responsive design and React for building dynamic user interfaces.

Backend Development

To manage server-side development, I learned PHP and Node.js, allowing me to create robust backend solutions. I further developed my database skills with SQL and NoSQL databases like MySQL and MongoDB, essential for data management in full-stack applications.

Tools of the Trade

Version control tools, particularly Git, became vital in my development workflow, enabling collaboration and efficient code management. I also utilized software tools like Visual Studio Code for coding, Postman for API testing, and Figma for UI/UX design, enhancing my ability to deliver user-centric applications.

Continuous Growth

My education and hands-on projects equipped me with the skills necessary to build fully developed websites, from front-end design to back-end integration. This journey has ignited my passion for continuous learning and adapting to new technologies, shaping me into a well-rounded developer ready to tackle real-world challenges in the software industry.

Projects

  • Project 1: Personal Portfolio Website

    This project involved creating a personal portfolio website to showcase my skills, projects, and experiences. The goal was to design an interactive and visually appealing site that would serve as an online resume. Technologies used: HTML, CSS, JavaScript.

    I used HTML for structuring the content, ensuring it was accessible. With CSS, I created a responsive design using Flexbox and Grid, and incorporated animations to enhance user engagement. JavaScript added interactivity, featuring a dynamic project gallery and smooth scrolling navigation.

  • Project 2: Task Management Application

    I developed a task management application that allows users to create, update, and delete tasks, helping them organize their daily activities. Technologies used: React, Node.js.

    Using React, I built a responsive interface with components for task lists and management. The backend was developed with Node.js and Express, enabling CRUD operations and user authentication, while data was stored in a MongoDB database.

  • Project 3: E-commerce Website

    This project focused on building a fully functional e-commerce website where users can browse products, add them to their cart, and make purchases. Technologies used: Python, Django.

    Using Django, I developed a backend to manage products and user data, implemented product listings with filtering options, and created a shopping cart. I integrated a third-party payment gateway for secure transactions and learned about RESTful API design and security practices.

Modules Completed in BCom Information Systems

The BCom Information Systems program at the University of Johannesburg is designed to deepen knowledge and develop applied competence in analyzing, interpreting, and understanding information systems development processes. Below is a detailed overview of the modules I completed during my studies:

First Year

  • Business Management 1A (BMA11A1)

    This module introduced fundamental principles of business management, covering topics such as organizational structure, strategic planning, and leadership styles. I learned to analyze business environments and make informed managerial decisions.

  • Development Software 1A (DEV1A01)

    Focused on the basics of software development, this module taught me programming concepts using languages such as Java and Python. I gained hands-on experience in creating simple applications and understanding software development life cycles.

  • Information Systems 1A (ILS1A01)

    This module provided insights into the role of information systems in organizations, covering system design, implementation, and management. I learned to analyze information needs and propose solutions for improving business processes.

Second Year

  • Development Software 2A (DEV2A01)

    This module advanced my programming skills, introducing concepts such as object-oriented programming and software testing. I worked on collaborative projects to develop complex applications, emphasizing teamwork and project management.

  • Entrepreneurship 2A (ETS12A2)

    Focusing on the principles of entrepreneurship, this module taught me how to develop business plans, identify market opportunities, and innovate within the technology sector. I learned to apply entrepreneurial thinking to problem-solving in business contexts.

  • Information Systems 2A (ILS2A01)

    Building on the first year, this module delved deeper into information systems analysis and design methodologies. I learned to create detailed system requirements and design specifications for effective information systems.

Third Year

  • Entrepreneurship 3A (ETS13A3)

    This module continued to develop my entrepreneurial skills, focusing on startup management and innovation strategies. I engaged in practical projects to launch business ideas and presented them to a panel of experts.

  • Information Systems 3A (ILS3A01)

    In this module, I explored advanced topics in information systems, including data management and systems integration. I learned about emerging technologies and their impact on business operations.

  • Information Systems Project (PRO001)

    This year-long project involved applying all the knowledge and skills I acquired throughout the program to develop a comprehensive information system for a real-world client. I led a team, managed the project timeline, and ensured quality deliverables.
